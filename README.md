# Exercício de SQL: Filtragem de Filmes

Este repositório contém um exercício prático de SQL focado em contagem e filtragem de dados com múltiplas condições.

## Objetivo
O objetivo é contar quantos filmes únicos em língua inglesa, lançados entre 1990 e 1999, possuem as classificações indicativas 'G', 'PG' ou 'PG-13'.

## Estrutura do Projeto
Para manter a organização, dividi o código em dois arquivos:
* `schema.sql`: Contém a estrutura da tabela `films`.
* `query.sql`: Contém a resolução do exercício.

## Como executar
1. Primeiro, execute o script de `schema.sql` para criar a tabela.
2. Em seguida, utilize o script `query.sql` para realizar a consulta.
